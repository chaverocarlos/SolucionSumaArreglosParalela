# 🚀 Suma de Arreglos en Paralelo con OpenMP

Este repositorio contiene la implementación de un algoritmo en **C++** diseñado para realizar la suma de dos arreglos de manera **paralela**, utilizando la librería **OpenMP**.

## 🎯 Objetivo
Demostrar el uso de **memoria compartida** y **paralelismo de datos** mediante la distribución de iteraciones de un ciclo `for` entre múltiples hilos de procesamiento (*threads*).

## ⚙️ Características
- **Procesamiento paralelo**: Uso de la directiva `#pragma omp parallel for` para dividir la carga de trabajo.  

## 📋 Requisitos
- Compilador de C++ compatible con OpenMP (Microsoft Visual Studio).  
- En Visual Studio: habilitar la opción de **OpenMP** en la configuración del proyecto.
